메뉴판 만들기

1. 캠핑장 정보 - 바둑판 배열
 - 캠핑장 사진(DB)
 - 캠핑장 이름(DB)
 - 캠핑장 주소(DB)
 - 캠핑장 특징/주의사항(DB)
 - 찾아오는 길(간단히)

2. 지도(가제)
 - 캠핑장 지도
 - 캠핑장 좌표(DB)
 - 날씨

3. 예약/결제
 - 캠핑 날짜
 - 캠핑 장소
 - 캠핑 인원 : 성인/미성년자
 - 금액 산정
 - 결제 : 카카오페이

4. 사이드 메뉴/도구 - html로 끝내기
 - 메뉴 : 간단한 간식류
 - 도구 : 간단한 도구들

5. 게시판
 - 공지사항 : 관리자
 - QnA : 관리자/사용자
 - 후기 : 사용자 - 사진첨부가능

*회원정보 users
 - 로그인
	- 아이디, 비밀번호 입력받아서
	- 네이버/카카오 연동 로그인?
 - 회원가입 : 자체(네이버/카카오톡 택 1) - 휴대폰인증 X
       - 필수정보
	- 본명(DB) user_name
	- 아이디(DB) user_id
	- 비밀번호(DB) user_pwd
	- 이메일(DB) user_email
	- 주민등록번호(DB) user_code
       - 선택정보
	- 나이(DB)
	- 성별(DB)
 - 마이페이지 : 로그인 해야 접속가능
       - 개인정보 수정
	- 아이디/비밀번호 변경
       - 관심 여행지(즐겨찾기/북마크)
 - 아이디/비밀번호 찾기
       - 아이디 : 이름과 주민번호 입력받으면 아이디 출력
       - 비밀번호 : 이름과 아이디 입력받으면 비밀번호 출력
